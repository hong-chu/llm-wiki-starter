# derived/

Optional outputs the agent can build *from* the wiki — charts, slide
decks, study sheets, exports. Treat this folder like `wiki/`: the
agent owns it, and you can blow it away and rebuild any time.

Examples to ask for once you have content:
- "Make a 1-page study sheet of every concept in `concepts/`."
- "Build a timeline chart from all `entities/*.md` `Timeline` sections."
- "Export a `.pptx` summarizing the `syntheses/` folder."
